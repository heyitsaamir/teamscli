# TypeScript Teams Bot with Tab

This is a Microsoft Teams bot template that includes a personal tab served by Vite.

## Structure

- `src/index.ts`: Main bot application code.
- `src/Tab/`: React app for the personal tab, bundled by Vite.
- `appPackage/`: Teams app manifest and icons (generated by the CLI).

## Static Tab

The generated manifest includes an extra static tab entry (`/tabs/test`) that points to the Vite dev server. This allows the bot to serve a personal tab alongside conversational features.

## Getting Started

1. Install dependencies: `npm install`
2. Run the app: `npm start`
